# Alex Berman - Strategy Notes

## Source Video
- Title: How To Write Killer Cold Emails That GUARANTEE Responses
- URL: https://www.youtube.com/watch?v=881Dr4lMey4

## Core Strategy Summary
1. Start with a personalized hook that proves the message is specifically for the recipient.
2. Keep body copy short and outcome-centric with one clear value proposition.
3. Add social proof (clients, use case, or measurable result) to reduce trust friction.
4. Use direct CTAs that are easy to answer quickly (yes/no or short scheduling prompt).
5. Build follow-up sequences that add fresh context instead of repeating the same message.

## B2B SaaS Application
- Use persona-based templates with lightweight personalization fields.
- Split test hooks and proof statements by vertical.
- Optimize around positive replies and meetings booked.
